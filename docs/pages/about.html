О лабораторной работе №7

Изучение систем контроля версий и веб-технологий

Цели работы

  • Освоить основные принципы работы с системой контроля версий Git
  • Научиться создавать и управлять локальными репозиториями
  • Изучить работу с ветками и слиянием изменений
  • Освоить взаимодействие с удаленными репозиториями (GitHub)
  • Научиться размещать статические сайты на GitHub Pages
  • Применить полученные знания на практике, создав многостраничный сайт

Задачи работы

  1. Установить Git и настроить рабочее окружение
  2. Создать локальный репозиторий для проекта
  3. Освоить основные команды Git (add, commit, status, log)
  4. Поработать с ветками (создание, переключение, слияние)
  5. Настроить подключение к GitHub
  6. Создать статический веб-сайт с несколькими страницами
  7. Разместить сайт на GitHub Pages
  8. Оформить подробный отчет о проделанной работе
