Примеры кода Git и HTML

Реальные команды и код, использованные в работе

Настройка Git


# Настройка пользователя
git config --global user.name "Veronika"
git config --global user.email "nazveronia@gmail.com"

# Проверка настроек
git config --global --list

# Настройка основной ветки
git config --global init.defaultBranch main

Первоначальная настройка Git перед началом работы

Создание репозитория


# Создание папки проекта
mkdir lab7_practice
cd lab7_practice

# Инициализация Git репозитория
git init

# Проверка состояния
git status

Создание нового Git репозитория

Работа с коммитами


# Добавление файлов
git add report.md
git add .

# Создание коммита
git commit -m "Добавлен отчет по лабораторной работе"

# Просмотр истории
git log --oneline
git log --graph --all

Основной рабочий цикл: добавление и коммит изменений

Работа с ветками


# Создание новой ветки
git branch detailed-report
git checkout detailed-report

# Или одной командой
git checkout -b detailed-report

# Переключение между ветками
git checkout main
git checkout detailed-report

# Слияние веток
git checkout main
git merge detailed-report

Создание, переключение и слияние веток
